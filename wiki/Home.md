# Data Mining:  a Leaner Approach

It turns out that data mining is easy if you follow three simple rules:

1. Find some crap;
2. Cut the crap;
3.  Go to step 1.

## Installation

LEANER uses a standard UNIX environment (with git, make, python 2.7+, bash, awk, etc).

Share and enjoy.
